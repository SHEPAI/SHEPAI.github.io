Arduino Project


CLIVE the Robot

CLIVE is a project started in 2019, which is based on the Arduino. The Arduino was used to experiment more with an embedded system, rather than using operating systems which are pre-made with their own behavoirs. The Arduino is also small and cheap to prototype and parallel manage systematic processes.
