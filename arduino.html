Arduino Clive

CLIVE is a project started in 2019, which is based on the Arduino. The Arduino was used to experiment more with an embedded system, rather than using operating systems which are pre-made with their own behavoirs. The Arduino is also small and cheap to prototype and parallel manage systematic processes.

The idea behind CLIVE is a system which constantly monitors it's enviroment, and builds up patterns of behaviors it notices. Through conditioning and repetition, CLIVE will be able to learn instructions. For simplicity, the project uses a pre-made object recognition camera, audio sensor and ultrasonic range finder. They take in data, and link simularities into blocks which can form larger actions. Compared to SHEP pre V0.0.7, CLIVE is auto learning rather than being told exactly how to respond.
