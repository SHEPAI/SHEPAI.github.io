Astromech project

In the past we have developed robotic systems with either complete learning, or preset abilities with no learning. In this project I will be working on a system which provides both. This system will have preset abilities to make it useful, while deploying different mechanisms of machine learning to adapt its current abilities, or even learn knew ones. It is the aim to make a system which is quickly deploy-able in a useful way, and can adapt to its environment. To demonstrate such a system of hybrid intelligence, this project is to develop a Star Wars Astromech droid.
