Hardware library


What is it

The hardware library is currently in development. SHEP will use Servos to move around. The library will be set up to control a series of robotic chassis. You could consider this the driver. It will be called in to the main Python project. A chassis type will be called in, and will manage it's balance and tasks in the background, while the AI main code loops round. This uses parallel programing, with multiple threads of data.

Here is an example of how the library will look.
from Robot_hardware import chassis1
robotInterface = chassis1("standing")
#create a chassis to run in the background, starting standing up robotInterface.walk()
robotInterface.goTo("location1")
