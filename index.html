Introduction


SHEP AI

Advancing the evolution

The AI matrix is a system which collects an input, checks its data and outputs the found string of words to the user. If the input is not found in the data, the AI will ask the user to add it. The input will then be in the solid state memory and accessible in the future when the AI is faced with this input. The output can be from saying words to physically learning through potentiometers and servos. SHEP has a network of files where an input is checked word for word to find an appropriete reply to the user.
