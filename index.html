Introduction


SHEP AI

SHEP is an Artificial intelligence development project, started 2015. The aim is to make an open source platform which can apply Artificial intelligence to businesses, developers, public services and people with no technical understanding.

SHEP has been in development for many years. The project started off a keyboard input Arduino board, using an SD card and text file to store data. SHEP moved on to be voice controlled on the Arduino, using a series of boards communicating with eachother as a network. Later on, SHEP became a python program on the Raspberry Pi. Different inputs were used with this program from audio to network cables.

A python library for SHEP was made so SHEP could be used to control any system, with ease. Different products are being developed using this new method. For version 0.0.6 we have SHEP using the internet to enhance it's knowledge.

The latest release (hopefully to be released by the end of the year) will add input analysis skills, to fully learn a language and understand what it is saying. This will enhance SHEP's ability to interact and intergrate into any task. This is being developed on the Arduino, but a version will be released as a Python library. This opens it up to developers.

SHEP has been in development for 5 years now, from a small arduino board to a series of networks. SHEP is still experimental and is being adapted constantly. We ask that any distribution of SHEP's software leave credit to the original developers.

CLIVE


CLIVE, the cognitive learning intelligent virtual enviroment is the latest proto-type project on the Arduino to test out artificial neural networks. It is designed to monitor and save information from its enviroment, then learn natrually from monitoring associations of data. This project is about understanding the brain and reproducing it into a computer system.
