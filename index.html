The Project

SHEP is an Artificial intelligence development project, started as a hobby in 2015. The aim is to make an open source platform which can apply Artificial intelligence to real world problems.

SHEP has been in development for many years. The project started off a keyboard input Arduino board, using an SD card and text file to store data. SHEP moved on to be voice controlled on the Arduino, using a series of boards communicating with eachother as a network. Later on, SHEP became a python program on the Raspberry Pi. Different inputs were used with this program from audio to network cables.

A python library for SHEP was made so SHEP could be used to control any system, with ease. Different products are being developed using this new method. For version 0.0.6 we have SHEP using the internet to enhance it's knowledge.

SHEP has undergone many changes in projects and design. Natural language adaptive chatbots were a key focus for a number of years, in order to make self learning interactive robots. The latest versions of this can be seen in libraries. The latest focus is on bio-inspired AI, specifically insect inspired robotics.

SHEP has been in development for over 7 years now, from a small arduino board to a series of networks. SHEP is still experimental and is being adapted constantly. We ask that any distribution of SHEP's software leave credit to the original developers.
