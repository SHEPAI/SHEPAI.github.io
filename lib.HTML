Python library


Python library

The Python library uses the newest V0.0.5 code. It works with a text interface, but can be modified for the user's preferences in the file.
The AI can be used by the user in Python. This means your projects can be intelligent learning systems. SHEP is created as an object Download the library to get started! The file "SHEP.py" will need to be in the same folder as your code. For the SHEP library to run, you will need to use any Python 3 and have certain libraries installed.

Libraries
  • lxml
  • urllib
  • bs4
Using SHEP
You can include the SHEP library using the following line:
from SHEP import AI

The library has several functions the user can use.

class = AI(AI_name,user_owner,datafile) -- This declares the AI with it's name and the name of the user. The datafile must be a string and an xml file

string = class.search(string) -- will look for the keywords in the AI then if they are present, check for an output. If nothing is present, it will return whats missing.

class.addword(string_to_add,string_of_word) -- this adds the string_to_add to the vocabulary file. The type of word is found automatically in the find_term()

string = class.find(list_of_keywords) -- This code will search the data for the words entered.

string = class.find_term(list_of_keywords) -- This code will search the data for the words entered.

string = class.edit("sentence","repleacement output") -- Edit a currently saved sentence. Will return /00/00/00 if the sentence is not currently saved. Will ask for replacement if setence is found and will save.

list = class.wordAdder("sentence",) -- This breaks up a sentence and returns a 2D array containing vocab

class.update() -- updates the code to the latest version

class.PutIn(string) -- This function provides a simple input, and returns what you type. This can be edited to accept different inputs.

class.out(string) -- Output as a string, but can be changed.

class.setFile(string) -- Change the datafile to a different one

class.setpath(string) -- Change the pathway to the files. The user may want to access a certain folder of data

string = class.learn(list_of_keywords,"word to add") -- Asks for an input and saves it. Returns the saved value message

string = class.addAction("file name") -- Asks for a input and returns whether or not it has saved

string = class.research() -- Check internet for information and returns either the link or nothing

class.listOfVocab -- This variable is a list of all the keywords in the sentence.

class.listUSB() -- check all USB ports and copy over files into the action folder. Take note certain modules will need to be installed. This only works with Linux




Here is an example of searching the AI with user input, manual adding words to the data, asking the AI using a string and editing current words:
from SHEP import AI
myBot = AI("Jeff", "user","knowledge.xml") #an AI called 'Jeff' and owned by 'user'
User = input("Your message ")
r = myBot.search(User)
print(r)
#add phrases to make word
myBot.addWord("how","t")
myBot.addWord("feeling","s")
myBot.addWord("you","c")

print("Robot message: "+myBot.find("how","feeling","you")) #check data
print("Robot message: "+myBot.find("no word","no word","no word")) #check data
#put the 1 as a 0 to enable automatic learning

myBot.edit("what is your name")
