Python SHEP library

SHEP uses a neural network graph to store information. The library provides the ability to sort out data entered by the user. The data is sorted by what it is entered with, and what data follows it. Using weighted values, the strength of connections is defined, which can increase when found together, and decrease when not. The more inputs, the clearer correlations in data becomes.

SHEP is designed to take in input data. If your robot project has an array of different sensors, the values can be entered into SHEP and tasks learned. Everytime A robot hears me clap my hands I might want it to play a sound back. SHEP will work out that everytime I clap I make a noise after, which means SHEP would return the value of what I want it to say everytime I clap.

Download the library to get started! The file "SHEP.py" will need to be in the same folder as your code.

For the SHEP library to run, you will need to use any Python 3.



Using SHEP
You can include the SHEP library using the following line:
from SHEP import AI

The library has several functions the user can use.


class = AI(system pathway) -- This declares the AI with it's pathway to it's files

string_list = class.process(string_list_of_inputs) -- This finds the relevant items of data which match inputs

string_list = class.enterData(string_list_of_inputs,type) -- This finds the relevant items of data, the type is the connection type as a string (SNT and DNT)

string_list = class.getConnected(string_list_of_inputs,type) -- Find data which is strongly connected to parameters

class.link(string_of_items,items_to_link_to) -- Change/set the pathway during the program

class.negFeedback(string_list_of_inputs, string_list_to_lower) -- This code will search the data for possible outputs


string_list_of_inputs=["12,32,43,12","dog.jpg","sensorVal=23"]
stringtypes=["sound","sight",""]


Here is an example of searching the AI with user input, manual adding words to the data, asking the AI using a string and editing current words:
from SHEP import AI
#This code represents input arrays from objects a camera system sees
bot=AI("test/testFiles/newGraph/")#Where I want the data stored
print(bot.process(["cat","hand","apple"]))
print(bot.process(["pat cat","orange","dog"]))
print(bot.process(["cat",hand","soup"]))
print(bot.process(["pat cat","bannana","apple"]))
print(bot.process(["cat","rake","hand"]))
print(bot.process(["pat cat","pear","snake"]))
