Python library

The Python library uses the newest version of the AI software. It works with a text interface, but can be modified for the user's preferences.

The AI can be used for many of your projects to give it better automatic intelligence. Chose your own inputs in your code to send data and let SHEP's algorithm sort it.
The AI can be used by the user in Python. The current version is still in progress. SHEP is created as an object, and used in certain ways.

Download the library to get started! The file "SHEP.py" will need to be in the same folder as your code.

For the SHEP library to run, you will need to use any Python 3 and have certain libraries installed.



Using SHEP
You can include the SHEP library using the following line:
from SHEP import *

The library has several functions the user can use.


class = AI(system pathway, audioThreshold=2) -- This declares the AI with it's pathway to it's files, and the number of inputs

string_list = class.main(string_list_of_inputs,stringtypes) -- This finds the relevant items of data and stringtypes is optional

class.setPath(pathwayString) -- Change/set the pathway during the program

stringList = class.findValue(string_list_of_inputs,stringtypes) -- This code will search the data for possible outputs

class.negFeedback(string_output_given_from_find, list_of_nodes) -- This code will search the data for possible outputs


stringtypes is an array the size of string_list_of_inputs which specify what each element of string_list_of_inputs do

string_list_of_inputs=["12,32,43,12","dog.jpg","sensorVal=23"]
stringtypes=["sound","sight",""]


Here is an example of searching the AI with user input, manual adding words to the data, asking the AI using a string and editing current words:
from SHEP import AI
#This code represents input arrays from objects a camera system sees
myBot = AI("") #an AI called 'Jeff' and owned by 'user'
inputs=["Blue","Blue cup","dog"]
myBot.setPath("test/")
myBot.main(inputs)
inputs=["Blue","Blue cup","cat"]
myBot.main(inputs)
inputs=["cat","milk","dog"]
myBot.main(inputs)
inputs=["Blue","Blue cup","mat"]
myBot.main(inputs)
inputs=["cat","milk","popping candy"]
myBot.main(inputs)
