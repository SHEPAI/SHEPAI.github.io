Troubleshooting

You can use an HDMI screen to see what SHEP is thinking. Use this to backtrack where SHEP isn’t understanding

Action adding will not add

make sure you don’t use capital letters in the file name, and call it something SHEP can understand.

Voice input isn't listening

Voice input – be within a 3m range of SHEP when talking. Make sure the sound enviroment isn't too loud. SHEP mostly adapts to the room sound, but if the volume is incredibly loud the microphones will not pick up your voice.

Voice input cuts out too quickly

I change my mind when adding or just simply talking

When you are talking to SHEP, whether adding a phrase, teaching SHEP infomation, or simply asking a question, you may want to stop. SHEP can stop if you simply say "cancel". If cancel does'nt work (due probably to pronounciation) try "exit".

SHEP is slow to respond

We continue to work on this to try increase speed. Sadly this is the fastest we can make SHEP.

SHEP does not respond accurately

SHEP uses the internet to search for answers. If an answer is not correct, you can edit it using "robot edit". See tutorial
