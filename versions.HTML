Versions

The best way to stay consistent is to change with the circumstances - Winston Churchill

Type one:
-V1.0 = This was the learning algorithm code using text input on the Arduino Uno
-V1.1 = This was the updated version with audio input and output. This version also possesed artificial emotions.
-V1.2 = This version had servo output and an updated voice input. This version also possesed a touchscreen interface
These versions had their problems such as; audio not working, not enough RAM, wiring errors. These bugs were fixed but it was decided SHEP needs to be updates.

Type two:
-AI_2 = This algorithm spread across multiple boards, the user could decide on their own input and wire i2c channels to the main AI board.
This version was bulky and had alot of wires coming out consistently.

Type three:
-V0.0.1 = This version used a desktop computer on Python, it had better audio i/o capibilities and the ability to output through servos. this version creates a nueral network of files.
-V0.0.2 = This version is on the Raspberry Pi and has an interactive GUI, an understanding of the english language, the ability to output through servos and text input if there is no internet connection.

Below shows downloads to older versions and how to make them. These are not very advanced AIs, more of theory testers.
-V0.0.3 = This moved to a nicer UI. The AI could respond based on the topic of conversation, and split long sentences up into smaller ones. Volitile subjects were introduced, and new layers of vocabulary.
-V0.0.4 = The ability to play actions was introduced. Voice recognition was taken away, and text input re-introduced. A text to speech USB input was developed.
-V0.0.5 = This version was to scrap a GUI, and run using voice input and output only.

Type four
-V0.0.6 = This version pushed the AI into the world of object oriented code. This opened up its abilities, and made it easier to code and adapt to solve problems. It also introduced automatic subject adding
-V0.1.1 = This version is the SHEP AI library for V0.0.6
-V0.0.7 = This version got rid of any sort of manual vocabulary adding, and automatically read and saved words. Multiple keywords can be added to a phrase being saved, to increase accuracy. An internet search is done to answer the user's questions, and links saved so the AI can quickly answer them again.
