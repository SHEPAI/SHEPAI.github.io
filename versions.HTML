Versions

The best way to stay consistent is to change with the circumstances - Winston Churchill

Type one:
-V1.0 = This was the learning algorithm code using text input on the Arduino Uno
-V1.1 = This was the updated version with audio input and output. This version also possesed artificial emotions.
-V1.2 = This version had servo output and an updated voice input. This version also possesed a touchscreen interface
These versions had their problems such as; audio not working, not enough RAM, wiring errors. These bugs were fixed but it was decided SHEP needs to be updates.

Type two:
-AI_2 = This algorithm spread across multiple boards, the user could decide on their own input and wire i2c channels to the main AI board.
This version was bulky and had alot of wires comming out consistently.

Type three:
-V0.0.1 = This version used a desktop computer on Python, it had better audio i/o capibilities and the ability to output through servos. this version creates a nueral network of files.
-V0.0.2 = This version is on the Raspberry Pi and has an interactive GUI, an understanding of the english language, the ability to output through servos and text input if there is no internet connection.
